2

Enable BigQuery Billing Export

Export billing data to BigQuery to track costs in XamOps dashboard.

1. Go to Billing Export:
  • Open GCP Console → Billing
  • Select your billing account
  • Go to Billing export → BigQuery export
2. Enable Standard Usage Cost Export:
  • Click Edit settings for "Standard usage cost"
  • Select project: YOUR_PROJECT_ID
  • Create or select dataset: billing_export
  • Click Save
3. Note the table name (format):
PROJECT_ID.billing_export.gcp_billing_export_v1_XXXXXX_XXXXXX_XXXXXX

You'll need this exact table name later in the form below.

3

Grant BigQuery Dataset Access

Give XamOps service account permission to read billing data.

Steps:
  • Go to BigQuery Console
  • Find your billing_export dataset
  • Click the ⋮ menu → Share → Permissions
  • Click Add Principal
  • Enter service account: xamops@YOUR_PROJECT_ID.iam.gserviceaccount.com
  • Select role: BigQuery Data Viewer
  • Click Save
4

Verify Permissions (Optional)

Confirm all permissions are correctly set.

Check Service Account Roles:
gcloud projects get-iam-policy YOUR_PROJECT_ID \
  --flatten="bindings[].members" \
  --filter="bindings.members:xamops@*"
Expected roles:
  • roles/viewer - Read access to all resources
  • roles/monitoring.viewer - Read metrics and monitoring data
